## Dependencies

- github.com/nlopes/slack

```

```

- gopkg.in/yaml.v2

```
go get -u gopkg.in/yaml.v2
```

- github.com/perterhellberg/giphy

```
go get -u github.com/peterhellberg/giphy
```

```
go build
```
